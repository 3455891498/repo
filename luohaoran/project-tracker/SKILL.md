# 项目进度跟踪

## 功能概述

帮助用户跟踪和管理项目进度，通过询问关键信息生成结构化的项目进度报告和任务清单。

## 工作流程

1. **收集项目基本信息**：项目名称、目标、截止日期
2. **识别关键任务**：分解项目为可跟踪的任务单元
3. **定义任务状态**：未开始、进行中、已完成、阻塞
4. **生成进度报告**：可视化展示项目进展情况

## 输出文件

| 文件 | 一句话定位 | 解决什么问题 |
| --- | --- | --- |
| `project-info.md` | 项目基本信息 | 记录项目核心信息，便于查阅 |
| `task-list.md` | 任务清单 | 清晰列出所有任务及其状态 |
| `progress-report.md` | 进度报告 | 可视化展示项目完成情况 |
| `blockers.md` | 阻塞问题记录 | 跟踪阻碍进度的问题 |

## 规范格式

### project-info.md

```
项目名称：[项目名称]
项目目标：[简要描述]
截止日期：[YYYY-MM-DD]
负责人：[负责人姓名]
状态：[进行中/已完成/暂停]
```

### task-list.md

```
## 任务清单

| 序号 | 任务名称 | 负责人 | 状态 | 截止日期 | 优先级 |
| --- | --- | --- | --- | --- | --- |
| 1 | [任务1] | [姓名] | [状态] | [日期] | [高/中/低] |
| 2 | [任务2] | [姓名] | [状态] | [日期] | [高/中/低] |
```

### progress-report.md

```
## 项目进度报告

### 概览
- 总任务数：[N]
- 已完成：[N] (XX%)
- 进行中：[N] (XX%)
- 未开始：[N] (XX%)
- 阻塞：[N] (XX%)

### 进度趋势
[使用 ASCII 图表展示]

### 风险提示
[列出潜在风险和建议]
```

### blockers.md

```
## 阻塞问题记录

| 序号 | 问题描述 | 影响任务 | 状态 | 计划解决时间 |
| --- | --- | --- | --- | --- |
| 1 | [问题描述] | [任务名称] | [待处理/处理中/已解决] | [日期] |
```

## 使用场景

- 团队项目进度跟踪
- 个人任务管理
- 项目状态汇报
- 风险预警和问题追踪
